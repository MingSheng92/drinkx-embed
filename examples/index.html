Drinkx Menu Embed Example

This is an example of how to use the Drinkx Menu Embed script on your website.

Method 1: Automatic Initialization

You can have the script automatically initialize by adding the data-menu-embed attribute to a container element. You must also provide a data-venue-id.

Note: Replace YOUR_VENUE_ID with your actual venue ID.

You can also initialize the script manually using JavaScript. This gives you more control over the options.
